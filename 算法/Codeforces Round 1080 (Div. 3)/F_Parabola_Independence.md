# **【算法题】F. Parabola Independence（抛物线独立性）**
每测试用例时间限制：3 秒
每测试用例内存限制：512 兆字节

### 题目描述
给定一个由 \(n\) 个二次函数组成的集合 \(F = \{f_1, f_2, \cdots, f_n\}\)，其中 \(f_i(x) = a_ix^2 + b_ix + c_i\) 。

两个函数 \(f\) 和 \(g\) 被称为相互独立，当且仅当对于所有 \(x \in R\) ，都有 \(f(x) \neq g(x)\) 。

同样，一个函数集合 \(G = \{g_1, g_2, \cdots, g_k\}\) 被称为有序的，当且仅当对于所有 \(1 \leq i < j \leq |G|\) ，函数 \(g_i\) 和 \(g_j\) 相互独立。

对于每个 \(i = 1, 2, \cdots, n\) ，请找出 \(F\) 中包含 \(f_i\) 作为元素的最大有序子集的大小。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 10^4\)）。接下来是测试用例的描述。

每个测试用例的第一行包含一个整数 \(n\)（\(1 \leq n \leq 3000\)）。

接下来 \(n\) 行，每行包含三个整数 \(a_i\)，\(b_i\)，\(c_i\) ，表示函数 \(f_i\)（\(-10^6 \leq a_i, b_i, c_i \leq 10^6\)，\(a_i \neq 0\)）。

保证在一个测试用例中的函数两两不同。

保证所有测试用例的 \(n^2\) 之和不超过 \(3000^2\) 。

### 输出
对于每个测试用例，输出 \(n\) 个整数 \(s_1, s_2, \cdots, s_n\) ，其中 \(s_i\) 是包含 \(f_i\) 的最大有序子集的大小。

### 示例
- **输入**
```
3
4
1 2 -1
-3 0 -3
-1 4 -5
1 2 -4
5
3 0 0
1 0 -5
-3 0 0
-1 0 10
1 0 -10
5
884 -667 497
680 -973 213
23 -548 -412
826 359 -333
773 212 218
```
- **输出**
```
3 2 3 3
3 3 2 2 3
3 3 3 1 2
```

### 说明
在第一个测试用例中，函数如下：
 - \(f_1(x) = x^2 + 2x - 1\) ；
 - \(f_2(x) = -3x^2 - 3\) ；
 - \(f_3(x) = -x^2 + 4x - 5\) ；
 - \(f_4(x) = x^2 + 2x - 4\) 。
这些函数的图像如下：
![alt text](https://espresso.codeforces.com/168b5d6d822586244824257fcad8369941df3285.png)


包含每个函数的 \(F\) 的最大有序子集如下：
 - \(\{f_1, f_3, f_4\}\) 是包含 \(f_1\) 的最大有序子集；
 - \(\{f_1, f_2\}\) 是包含 \(f_2\) 的最大有序子集；
 - \(\{f_1, f_3, f_4\}\) 是包含 \(f_3\) 的最大有序子集；
 - \(\{f_1, f_3, f_4\}\) 是包含 \(f_4\) 的最大有序子集。
