#Linux（常用）指令（markdown）
 --- 

COMMAND 0：
 - 指令名称：CTRL + D
 - 指令功能：中断信号（SIGINT）给当前正在前台运行的命令，强制终止执行
 --- 

COMMAND 1：
 - 指令名称：ssh
 - 指令功能：用于远程登录到其他主机上执行命令
 - 指令语法：ssh [选项] [用户名+@+ip地址]
 - 指令示例：ssh user@192.168.1.1
 - 指令说明：通过ssh协议登录到指定主机上，默认端口22。
--- 
COMMAND 2：
 - 指令名称：ls
 - 指令功能：显示目录下的文件及文件夹
 - 指令语法：ls [选项] [参数]
 - 指令示例：ls /home
 - 指令说明：列出指定目录下的文件和目录，默认显示当前目录。
 - 选项可选：
     - -a：显示所有文件，包括隐藏文件。
     - -l：以长格式显示文件和目录的详细信息。
     - -h：以易读的单位显示文件大小。  
---
COMMAND 3：
 - 指令名称: pwd
 - 指令功能：显示当前工作目录的路径
 - 指令语法：pwd
 - 指令示例：pwd
 - 指令说明：显示当前所在的目录路径。
 - 补充：pwd命令不会有任何选项或参数，它只会输出当前工作目录的绝对路径。  
---
COMMAND 4：
 - 指令名称：cd
 - 指令功能：切换当前工作目录
 - 指令语法：cd [目录路径]
 - 指令示例：cd /home/user
 - 指令说明：改变当前工作目录到指定的目录。
 - 补充：
     - ..：切换到上一级目录。
     - ~：切换到用户主目录。
     - -：切换到上一次访问的目录。
     - .：切换到当前目录。  
     - /：切换到根目录。
---
COMMAND 5：
 - 指令名称：touch
 - 指令功能：创建空文件或更新文件的时间戳
 - 指令语法：touch [选项] [文件名]
 - 指令示例：touch newfile.txt
 - 指令说明：如果指定的文件不存在，则创建一个空文件；如果文件已存在，则更新其访问和修改时间戳。
 - 选项可选：
     - -c：如果文件不存在，则不创建新文件。
     - -d：指定一个时间戳来设置文件的访问和修改时间。
 - 补充：
     - 可以使用多个文件名作为参数，同时创建多个空文件。
     - 可以使用绝对路径或相对路径指定文件位置。
 - 注意：
     - 该指令只能创建空文件，不能创建目录。
     - 该指令只能改变文件的访问和修改时间戳，不能改变文件的内容。
---
COMMAND 6：
 - 指令名称：echo
 - 指令功能：显示文本或变量的值
 - 指令语法：echo [选项] ["文本或变量"]   ps:还是推荐加上双引号比较好。
 - 指令示例：echo "Hello, world!"
 - 指令说明：将指定的文本或变量的值输出到标准输出（终端）。
 - 选项可选：
     - -n：不输出换行符。
     - -e：解释反斜线转义字符。（例如：\n表示换行，\t表示制表符）
     - --help：显示帮助信息。
     - --version：显示版本信息。
 - 补充：
     - 启动转义：  
       使用-e来转义特殊功能  
       使用反斜线“\”来转义特殊字符
        ```
        输入为：
        echo \"Hello, world!\"

        输出：
        "Hello, world!"（包含了双引号）
        ```
     - 使用双引号或单引号来包含文本。
        ```
        输入为：
        echo "
        > hello world（这里的“> ”是系统自带的）
        > belgnas"

        这个时候输出就为：（输出会保留换行和格式）
        （这行是一个换行）
        hello world
        belgnas
        ```
     - 彩色输出：使用 ANSI 转义码 \033[XXm
        ```
        输入为：
        echo -e "\033[31mHello, world!\033[0m"

        输出为：红色的“Hello, world!”
        ```
     - 输出重定向
         - \> ：覆盖文件
         - \>> ：追加到文件
        ```
        输入为：
        echo "Hello, world!" > output.txt（覆盖文件）
        echo "Hello again!" >> output.txt（追加到文件）
        ```
