# **【算法题】K. String Divide II（字符串分割II）**
每测试用例时间限制：15 秒
每测试用例内存限制：512 兆字节

### 题目描述
给定一个长度为 \(n\) 的字符串 \(s = s_1s_2\cdots s_n\)，该字符串仅由小写英文字母组成。为方便起见，我们定义字符串 \(s[l, r] = s_ls_{l + 1}\cdots s_r\)，即 \(s\) 从索引 \(l\) 到 \(r\) 的子字符串。

给定一个整数 \(k\)，你对由 \(k\) 个相同字符串连接而成的 \(s\) 的子字符串感到好奇。请找出这些子字符串的最长长度。

正式地说，你需要找到一组区间 \(\{[l_1, r_1]\)，\([l_2, r_2]\)，\(\cdots\)，\([l_k, r_k]\}\)，满足以下两个条件：
 - 对于每个 \(i\)（\(1 \leq i \leq k\)），\(1 \leq l_i \leq r_i \leq n\)；
 - 对于每个 \(i\)（\(1 \leq i < k\)），\(r_{i + 1} = l_{i + 1}\)，且 \(s[l_i, r_i] = s[l_{i + 1}, r_{i + 1}]\)。

你需要使 \(r_k - l_1 + 1\) 最大化。

### 输入
第一行包含两个整数 \(n\)，\(k\)（\(2 \leq k \leq n \leq 10^6\)）。

第二行包含一个长度为 \(n\) 的字符串 \(s\)，仅由小写英文字母组成。

### 输出
输出一个整数，表示答案。特别地，如果不存在这样的子字符串，则输出 \(0\)。

### 示例
- **输入**
```
5 3
bacbc
```
- **输出**
```
0
```
- **输入**
```
7 2
ababbba
```
- **输出**
```
4
```

### 说明
对于第二个示例，我们选择的区间集是 \(\{[1, 2], [3, 4]\}\)。可以证明这是满足条件的最长子字符串。
