# **【算法题】【模板】拓扑排序**

时间限制：1 秒
空间限制：256M

### 描述
给定一张包含 \(n\) 个顶点、\(m\) 条边的有向图，请输出其拓扑序（Topological Order）。若存在多个合法序列，可输出任意一种；若图中存在有向环，无法拓扑排序，则输出 \(-1\)。

**【名词解释】**：拓扑序（Topological Order）是指有向无环图（DAG）的所有顶点的一个线性排列（记为 \(v_1, v_2, \cdots, v_n\)），满足对于图中任意一条从顶点 \(v_i\) 到顶点 \(v_j\) 的有向边，在拓扑序中，\(v_i\) 必须出现在 \(v_j\) 的前面。即，如果存在边 \((v_i, v_j) \in E\)，则在排列中 \(v_i\) 的索引 \(i\) 必须小于 \(v_j\) 的索引 \(j\)。

### 输入描述
第一行输入两个整数 \(n, m\) \((1\leq n, m\leq 2\times10^{5})\) —— 顶点数与边数。

接下来 \(m\) 行，每行输入两个整数 \(u_i, v_i\) \((1\leq u_i, v_i\leq n)\)，表示一条从 \(u_i\) 指向 \(v_i\) 的有向边。

### 输出描述
若图存在拓扑序，输出一行 \(n\) 个整数，按照拓扑顺序依次给出顶点编号（相邻数字以空格分隔，行尾不得有多余空格）。

若图无拓扑序，输出一行一个整数 \(-1\)，表示无解。

### 示例1
- **输入**
```
5 4
1 2
2 3
3 4
4 5
```
- **输出**
```
1 2 3 4 5
```
