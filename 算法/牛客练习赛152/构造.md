# **【算法题】构造**
时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
空间限制：C/C++/Rust/Pascal 256 M，其他语言512 M
Special Judge, 64bit IO Format: %lld

### 题目描述 
给定长度为 \(n\) 的数组 \(a_1,a_2,\ldots,a_n\) ，现在需要构造一个长度为 \(n\) 的数组 \(b_1,b_2,\ldots,b_n\) ，使其满足：
 - 对于任意的 \(i(1\leq i\leq n)\)，都有 \(a_i = mex(b_1,b_2,\ldots,b_{i - 1},b_{i + 1},\ldots,b_n)\) 。
 - 数组 \(b\) 中的元素满足 \(0\leq b_i\leq10^9\) 。

若不存在满足条件的数组 \(b\)，则输出 \(-1\) 。

### 名词解释
mex：整数数组的 \(mex\) 定义为未在数组中出现的最小非负整数。例如，\(mex(1,2,3)=0\) 、\(mex(0,2,5)=1\) ，若为空集，则 \(mex() = 0\) 。

### 输入描述
每个测试文件包含多组测试数据。第一行输入整数 \(T(1\leq T\leq10^4)\) ，代表数据组数，每组测试数据描述如下：

第一行输入整数 \(n(1\leq n\leq2×10^5)\) ，表示数组 \(a\) 的长度。
第二行输入 \(n\) 个整数 \(a_1,a_2,\ldots,a_n(0\leq a_i\leq10^9)\) ，表示数组元素。

此外，保证单个测试文件中所有 \(n\) 的总和不超过 \(2×10^5\) 。

### 输出描述
对于每组测试数据，另起一行。若能构造出满足条件的数组 \(b\)，输出 \(n\) 个整数，表示构造的数组 \(b\)；否则，输出 \(-1\) 。

若存在多个解决方案，可输出任意一个，系统会自动判定其正确性。注意，自测运行功能可能因此返回错误结果，请自行检查答案的正确性。

### 示例1
- **输入**
```
4
2
1 0
4
0 1 2 3
6

3

```
- **输出**
```

0 1 2 3

```
