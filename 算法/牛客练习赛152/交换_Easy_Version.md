# **【算法题】交换（Easy Version）**
时间限制：C/C++/Rust/Pascal 2秒，其他语言4秒
空间限制：C/C++/Rust/Pascal 512 M，其他语言1024 M
64bit IO Format: %lld

### 题目描述 
本题与《E.交换（Hard Version）》共享部分题目背景，但求解内容不同。

给定一棵以 \(1\) 为根的树，共有 \(n\) 个节点（编号为 \(1\) 到 \(n\)），每个节点 \(i\) 有两个值，分别是权值 \(a_i\) 与限制 \(s_i\)，且 \(s_i\) 取值为 \(0\) 或 \(1\)。

现在你可以执行以下操作任意次（也可以不执行）：
- 选择一条父子边 \((p, u)\)（其中 \(p\) 是 \(u\) 的父亲），需满足 \(a_u \geq a_p\) ，然后交换节点 \(p\) 与节点 \(u\) 的权值（即 \(\mathrm{swap}(a_p, a_u)\)）。

称一个可由初始状态通过若干次操作达到的状态 \(b_1, b_2, \ldots, b_n\) 为“最终合法状态”，当且仅当：
- 对于每个节点 \(i (1 \leq i \leq n)\)，其最终权值 \(b_i\) 均满足 \(b_i \leq s_i\) 。
两种“最终合法状态” \(b_1, b_2, \ldots, b_n\) 与 \(b_1', b_2', \ldots, b_n'\) 不同，当且仅当存在某个位置 \(i (1 \leq i \leq n)\)，使得 \(b_i \neq b_i'\) 。

求解有多少种不同的“最终合法状态”。由于答案可能很大，请将答案对 \(998244353\) 取模后输出。

### 输入描述:
每个测试文件均包含多组测试数据。第一行输入一个整数 \(T(1 \leq T \leq 100)\) 代表数据组数，每组测试数据描述如下：

第一行输入一个整数 \(n (1 \leq n \leq 4×10^3)\)，表示节点数。
此后 \(n - 1\) 行，第 \(i\) 行输入两个整数 \(u_i, v_i (1 \leq u_i, v_i \leq n)\)，表示树上第 \(i\) 条无向边连接节点 \(u_i\) 和节点 \(v_i\) 。
第 \(n + 1\) 行输入 \(n\) 个整数 \(a_1, a_2, \ldots, a_n (0 \leq a_i \leq 1)\)，表示每个节点的权值。
第 \(n + 2\) 行输入 \(n\) 个整数 \(s_1, s_2, \ldots, s_n (0 \leq s_i \leq 1)\)，表示每个节点的限制。

除此之外，保证单个测试文件的 \(n\) 之和不超过 \(4×10^3\) 。

### 输出描述:
对于每一组测试数据，新起一行输出一个整数，表示所有不同的“最终合法状态”的个数对 \(998244353\) 取模后的结果。
