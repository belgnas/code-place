# **【算法题】C. Dice Roll Sequence（骰子滚动序列）**
每测试用例时间限制：2 秒
每测试用例内存限制：256 兆字节

### 题目描述
考虑如下正方体 \(D\)，其中数字 \(x\) 和 \(7 - x\) 位于相对面：
（此处文字描述中提及图片，但未提供图片链接，图片由Nano Banana Pro生成）

一个由 \(1\) 到 \(6\) 的整数组成的序列 \(b\)，若满足以下条件，则称为骰子滚动序列：
所有相邻元素对都位于正方体的相邻面上。
例如，\([1, 4, 2]\) 是一个骰子滚动序列，而 \([3, 4, 6, 3]\) 不是，因为 \(3\) 和 \(4\) 不在骰子的相邻面上。此外，\([2, 2, 4]\) 也不是骰子滚动序列，因为 \(2\) 和 \(2\) 在骰子的同一个（非相邻）面上。

给定一个由 \(n\) 个 \(1\) 到 \(6\) 的整数组成的序列 \(a\)，你可以执行以下操作任意次（可能为零次）：
选择一个索引 \(1 \leq i \leq n\) 以及一个整数 \(1 \leq x \leq 6\) 。然后，将 \(a_i\) 的值更改为 \(x\) 。
请确定将 \(a\) 变为骰子滚动序列所需的最少操作次数。

### 名词解释
正方体的两个面 \(S\) 和 \(T\) 被称为相邻，如果它们恰好共享正方体的一条棱。请注意，此条件也意味着 \(S \neq T\) 。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 10^4\)）。接下来是测试用例的描述。

每个测试用例的第一行包含一个整数 \(n\)（\(1 \leq n \leq 3 \cdot 10^5\)）。

每个测试用例的第二行包含 \(n\) 个整数 \(a_1, a_2, \cdots, a_n\)（\(1 \leq a_i \leq 6\)）。

保证所有测试用例的 \(n\) 之和不超过 \(3 \cdot 10^5\) 。

### 输出
对于每个测试用例，输出将 \(a\) 变为骰子滚动序列所需的最少操作次数。

### 示例
- **输入**
```
3
3
1 4 2
4
3 4 6 3
10
6 1 4 3 1 3 2 5 4 4
```
- **输出**
```
0
1
4
```

### 说明
对于第一个测试用例，序列 \(a\) 是 \([1, 4, 2]\) 。由于这已经是一个骰子滚动序列，答案为 \(0\) 。

对于第二个测试用例，序列 \(a\) 是 \([3, 4, 6, 3]\) 。
仅更改一个元素，你可以得到 \([3, 5, 6, 3]\) ，这是一个骰子滚动序列。

对于第三个测试用例，序列 \(a\) 是 \([6, 1, 4, 3, 1, 3, 2, 5, 4, 4]\) 。
更改恰好 \(4\) 个元素，你可以得到 \([5, 1, 4, 2, 1, 3, 2, 1, 5, 4]\) ，这是一个骰子滚动序列。
