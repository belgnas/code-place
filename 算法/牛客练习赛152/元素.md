# **【算法题】元素**
时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
空间限制：C/C++/Rust/Pascal 256 M，其他语言512 M
64bit IO Format: %lld

### 题目描述 
给定长度为 \(n\) 的数组 \(a_1, a_2, \ldots, a_n\)，现在你可以任意次执行以下操作之一：
- 选择两个不同下标 \(i, j\)（\(i \neq j\)），满足 \(a_i\) 与 \(a_j\) 奇偶性不同，然后：
    - 若 \(a_i < a_j\)，则 \(a_i := a_i + 1\) 并且 \(a_j := a_j - 1\)；
    - 若 \(a_i > a_j\)，则 \(a_i := a_i - 1\) 并且 \(a_j := a_j + 1\)。
- 选择两个不同下标 \(i, j\)（\(i \neq j\)），满足 \(|a_i|\) 与 \(|a_j|\) 互质，然后从四种操作中任选一种执行：
    - \(a_i := a_i + 1\)；
    - \(a_i := a_i - 1\)；
    - \(a_j := a_j + 1\)；
    - \(a_j := a_j - 1\)。

判断是否可以通过若干次操作（可以是零次）使得数组 \(a\) 中所有元素相同。

### 名词解释
互质：多个整数的最大公约数（gcd）如果恰好为 \(1\)，则称它们互质。例如，\(12\) 和 \(30\) 的公约数有 \(1, 2, 3, 6\)，其中最大公约数是 \(6\)，所以它们不互质；\(5\) 和 \(7\) 的公约数仅有 \(1\)，所以它们互质。特别地，整数 \(0\) 和任何整数都不互质。

### 输入描述
每个测试文件包含多组测试数据。第一行输入一个整数 \(T\)（\(1 \leq T \leq 10\)）代表数据组数，每组测试数据描述如下：
- 第一行输入一个整数 \(n\)（\(1 \leq n \leq 10^3\)），表示数组 \(a\) 的长度。
- 第二行输入 \(n\) 个整数 \(a_1, a_2, \ldots, a_n\)（\(1 \leq a_i \leq 10^9\)），表示数组元素。

此外，保证单个测试文件中 \(n\) 的总和不超过 \(10^3\)。

### 输出描述
对于每一组测试数据，另起一行。若可以通过若干次操作（可以是零次）使得数组 \(a\) 中所有元素相同，输出 \(YES\)；否则，输出 \(NO\)。

### 示例1
- **输入**
```
5
5
1 2 3 4 5
3
15 3 9
4
1 2 2 1
5
2 6 8 4 10
2
3 5
```
- **输出**
```
YES
NO
YES
NO
YES
```
