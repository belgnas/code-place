# **【算法题】排列**
时间限制：C/C++/Rust/Pascal 2秒，其他语言4秒
空间限制：C/C++/Rust/Pascal 1024 M，其他语言2048 M
64bit IO Format: %lld

### 题目描述 
给定整数 \(n\)，考虑一个长度为 \(n\) 的排列 \(p_1, p_2, \ldots, p_n\) 。令 \(D_t\) 为排列 \(p\) 的第 \(t\) 个前缀 \(\{p_1, p_2, \ldots, p_t\}\) 内的正序对数，即：

\(D_t = |\{(i, j) | 1 \leq i < j \leq t, p_i < p_j\}| \)

其中，\(| \{(i, j) | 条件\}| \) 表示满足条件的 \((i, j)\) 对数。

现在，共有 \(q\) 次询问，每次询问求满足 \(l_i \leq D_{t_i} \leq r_i\) 的“合法不同排列”的数目。由于答案可能很大，请将答案对 \(998244353\) 取模后输出。

### 名词解释
长度为 \(n\) 的排列：由 \(1, 2, \ldots, n\) 这 \(n\) 个整数按任意顺序组成的数组（每个整数均恰好出现一次）。例如，\(\{2, 3, 1, 5, 4\}\) 是一个长度为 \(5\) 的排列，而 \(\{1, 2, 2\}\) 和 \(\{1, 3, 4\}\) 都不是排列，因为前者存在重复元素，后者包含了超出范围的数。

### 输入描述:
第一行输入两个整数 \(n, q(1 \leq n \leq 10^6; 1 \leq q \leq 2×10^5)\)，表示排列 \(p\) 的长度、询问次数。
此后 \(q\) 行，第 \(i\) 行输入三个整数 \(t_i, l_i, r_i(1 \leq t_i \leq n; 0 \leq l_i \leq r_i \leq 2×t_i)\)，表示第 \(i\) 次询问的排列 \(p\) 的前缀长度、所询问区间。

除此之外，保证单个测试文件中，将 \(t_i\) 去重后的和不超过 \(5×10^5\) 。

### 输出描述:
在一行上输出 \(q\) 个整数，表示每一次询问的满足条件的“合法不同排列”的数目对 \(998244353\) 取模后的结果。

### 示例1
- **输入**
```
5 6
1 0 2
2 1 1
4 4 8
5 1 10
3 2 2
5 7 10
```
- **输出**
```
120 60 45 119 40 29
```

### 示例2
- **输入**
```
10 5
2 0 0

10 0 20
```
- **输出**
```
1814400 3326400 3024000 1623370 1319957
```

