# **【算法题】A. Painting With Two Colors（用两种颜色涂色）**
- **每个测试的时间限制**：1 秒
- **每个测试的内存限制**：256 兆字节

给定三个正整数 $n$，$a$ 和 $b$（$1 \leq a, b \leq n$）。

考虑一排 $n$ 个单元格，初始时全为白色，索引从 1 到 $n$。你将按顺序执行以下两个步骤：
1. 选择一个整数 $x$，使得 $1 \leq x \leq n - a + 1$，并将连续的 $a$ 个单元格 $x, x + 1, \ldots, x + a - 1$ 涂成红色。
2. 选择一个整数 $y$，使得 $1 \leq y \leq n - b + 1$，并将连续的 $b$ 个单元格 $y, y + 1, \ldots, y + b - 1$ 涂成蓝色。

如果一个单元格同时被涂成红色和蓝色，其最终颜色为蓝色。

如果对于从 1 到 $n$（包括 1 和 $n$）的每个整数 $i$，单元格 $i$ 的颜色与单元格 $(n + 1 - i)$ 的颜色相同，则认为网格的涂色是对称的。你的任务是确定是否存在整数 $x$ 和 $y$，使得网格的最终涂色是对称的。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 $t$（$1 \leq t \leq 500$）。接下来是测试用例的描述。

每个测试用例的第一行也是唯一一行包含三个整数 $n$，$a$ 和 $b$（$1 \leq n \leq 10^9$，$1 \leq a, b \leq n$）—— 网格的单元格数量以及每一步要涂色的单元格数量。

### 输出
对于每个测试用例，如果网格的最终涂色可能是对称的，则输出 “YES”；否则，输出 “NO”。

你可以以任何大小写形式输出答案。例如，字符串 “yEs”、“yes”、“Yes” 和 “YES” 都将被识别为肯定回答。

### 示例
**输入**
```
7
5 3 1
4 1 2
7 7 4
8 3 7
1 1 1
1000000000 1000000000 1000000000
3 2 1
```
**输出**
```
YES
YES
NO
NO
YES
YES
NO
```
