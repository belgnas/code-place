# **【算法题】E. Connected Components（连通分量）**
每测试用例时间限制：2 秒
每测试用例内存限制：512 兆字节

### 题目描述
在Eminor大陆上有 \(n\) 个王国，编号从 \(1\) 到 \(n\) 。每个王国都有两个属性值，分别用 \(a_i\) 和 \(b_i\) 表示。

当 \(a_i - a_j \leq i - j \leq b_i - b_j\) 或者 \(a_j - a_i \leq j - i \leq b_j - b_i\) 时，王国 \(i\) 和 \(j\)（\(i < j\)）之间有一条无向道路相连。

格夫（Gew）想知道在这片大陆上有多少个连通分量。

### 输入
第一行包含一个整数 \(n\)（\(1 \leq n \leq 10^6\)）。

接下来 \(n\) 行，第 \(i\) 行包含两个整数 \(a_i\)，\(b_i\)（\(-10^9 \leq a_i, b_i \leq 10^9\)）。

### 输出
输出一个整数，表示连通分量的数量。

### 示例
- **输入**
```
5
1 -4
3 -2
5 0
7 2
9 4
```
- **输出**
```
5
```
- **输入**
```
2
1 2
2 1
```
- **输出**
```
1
```
- **输入**
```
5
5 4
3 3
2 5
3 4
4 5
```
- **输出**
```
2
```
