# **【算法题】H. Games on the Ads 2: Painting（广告游戏2：涂色）**
每测试用例时间限制：5 秒
每测试用例内存限制：512 兆字节

### 题目描述
厌倦了广告里那些虚假游戏？现在来试试这个！

给定一个 \(n×n\) 的网格。对于每一行和每一列，都有一把与之关联的画笔，总共 \(2n\) 把画笔。当你选择使用一把画笔时，它会将其对应的行（或列）中的每个单元格涂成画笔指定的颜色。在游戏中，有 \(n\) 种颜色，编号从 \(1\) 到 \(n\) 。与行对应的画笔的指定颜色构成一个 \([1, 2, …, n]\) 的排列。同样，与列对应的画笔的颜色也构成一个 \([1, 2, …, n]\) 的排列。

![示例中的画笔和目标图案](https://espresso.codeforces.com/95f9a7b49a10c0ba165a8918bf365c590b6a6040.png)

网格最初为空，任何单元格都没有颜色。游戏的目标是将网格涂成给定的图案。

你不满足于仅判断是否能达到目标。你想知道，如果每把画笔恰好使用一次，在所有 \((2n)!\) 种可能性中，有多少种能得到目标图案？由于答案可能很大，输出对 \(998244353\) 取模的结果。

### 输入
第一行包含一个整数 \(n\)（\(1 \leq n \leq 20\)），表示网格的大小。

第二行包含 \(n\) 个整数 \(p_1, p_2, ⋯, p_n\)（\(1 \leq p_i \leq n\)），其中 \(p_i\) 表示与第 \(i\) 行对应的画笔的颜色。保证 \(p\) 是 \([1, 2, …, n]\) 的一个排列。

第三行包含 \(n\) 个整数 \(q_1, q_2, ⋯, q_n\)（\(1 \leq q_i \leq n\)），其中 \(q_i\) 表示与第 \(i\) 列对应的画笔的颜色。保证 \(q\) 是 \([1, 2, …, n]\) 的一个排列。

接下来 \(n\) 行，每行包含 \(n\) 个整数，表示一个 \(n×n\) 的矩阵 \(c\)（\(1 \leq c_{i,j} \leq n\)），其中 \(c_{i,j}\) 表示目标图案中第 \(i\) 行第 \(j\) 列单元格的颜色。

### 输出
输出一个整数，表示对 \(998244353\) 取模后的解的数量。

### 示例
- **输入**
```
2
1 2
1 2
1 1
2 2
```
- **输出**
```
6
```

### 说明
对于该示例，如果我们从上到下然后从左到右给 4 把画笔编号，达到目标图案的 6 种顺序如下：
 - \(3, 2, 4, 1\)
 - \(3, 4, 1, 2\)
 - \(3, 4, 2, 1\)
 - \(4, 1, 3, 2\)
 - \(4, 3, 1, 2\)
 - \(4, 3, 2, 1\)
