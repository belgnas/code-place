# **【算法题】D. The 67th OEIS Problem（第67个整数序列在线百科问题）**
每测试用例时间限制：2 秒
每测试用例内存限制：256 兆字节

### 题目描述
猕猴作为一个糟糕的出题人，有一天决定在整数序列在线百科（OEIS）\(^{*}\) 上搜索有趣的数列，这样他就能为他那注定失败的泛哺乳动物信息学奥林匹克竞赛（PMOI）出题工作获取灵感。令他高兴的是，他找到了一个，并且觉得让你，他忠诚的测试者，来解决这个问题会很有趣：

构造一个包含 \(n\) 个整数的序列 \(a\) ，使得对于所有 \(1 \leq i \leq n - 1\) ，\(\gcd(a_i, a_{i + 1})\) \(^{\dagger}\) 的值都不同。保证至少存在一个这样的序列 \(a\) 。

### 名词解释
 - \(^{*}\) 整数序列在线百科，是数学怪才、极其敏锐的测试者和不够严谨的协调者最喜欢的网站。
 - \(^{\dagger}\) \(\gcd(x, y)\) 指整数 \(x\) 和 \(y\) 的最大公约数。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 100\)）。接下来是测试用例的描述。

接下来的 \(t\) 行，每行包含一个整数 \(n\)（\(2 \leq n \leq 10^4\)）—— 期望构造的序列长度。

保证所有测试用例的 \(n\) 之和不超过 \(10^4\) 。

### 输出
对于每个查询，输出你的答案 —— 一个由 \(n\) 个用空格分隔的整数组成的序列 \(a\)（\(1 \leq a_i \leq 10^{18}\)）。

### 示例
- **输入**
```
2
3
5
```
- **输出**
```
1 6 2
134 67 69 207 414
```

### 说明
在第一个测试用例中，序列 \([1, 6, 2]\) 是一个可能的答案。这是因为 \(\gcd(1, 6)\) 不等于 \(\gcd(6, 2)\) 。

在第二个测试用例中，序列 \([134, 67, 69, 207, 414]\) 是一个可能的答案。这是因为对于所有 \(1\) 到 \(n - 1\) 之间的 \(i\) ，\(\gcd(a_i, a_{i + 1})\) 的值都不同。供参考，它们分别是 \(67\) 、\(1\) 、\(69\) 和 \(207\) 。
