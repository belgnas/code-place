# **【算法题】A. Maple and Multiplication（梅布尔与乘法运算）**
- **每个测试的时间限制**：1 秒
- **每个测试的内存限制**：256 兆字节

梅布尔有两个正整数 $a$ 和 $b$。她可以执行以下操作任意次（可能为零次），以使 $a$ 等于 $b$：

选择任意正整数 $x$，并将 $a$ 或 $b$ 乘以 $x$。

你的任务是确定使 $a$ 等于 $b$ 所需的最少操作次数。可以证明这总是可行的。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 $t$（$1 \leq t \leq 100$）。接下来是测试用例的描述。

每个测试用例的第一行也是唯一一行包含两个正整数 $a$ 和 $b$（$1 \leq a, b \leq 1000$）—— 梅布尔当前拥有的两个数。

### 输出
对于每个测试用例，输出一个整数，表示梅布尔使 $a$ 等于 $b$ 所需的最少操作次数。

### 示例
**输入**
```
3
1 2
10 3
1000 1000
```
**输出**
```
1
2
0
```

### 注意
在第一个测试用例中，你可以将 $a = 1$ 乘以 $2$，得到 $a = b = 2$。这需要一次操作。

在第二个测试用例中，你可以将 $a = 10$ 乘以 $300$，得到 $a = 3000$，然后将 $b = 3$ 乘以 $1000$，得到 $b = 3000$。这需要两次操作。注意，操作后数字可能会超过 $1000$。

在第三个测试用例中，$a$ 和 $b$ 已经相等，所以不需要任何操作。
