# **【算法题】D. Reserved Reversals**

**时间限制**: 2秒  
**内存限制**: 256 MB

## 题目描述

给定一个由 $n$ 个正整数组成的数组 $a$。

对于数组中任意以位置 $l$ 开始、以位置 $r$ 结束（$1 \le l \le r \le n$）的段，设 $m(l,r)$ 表示该段中的最小值，$M(l,r)$ 表示该段中的最大值。形式化地：

$$ m(l, r) = \min(a_l, a_{l+1}, \dots, a_r), $$
$$ M(l, r) = \max(a_l, a_{l+1}, \dots, a_r). $$

你可以执行以下操作任意次（包括零次）：

- 选择两个下标 $l$ 和 $r$（$1 \le l \le r \le n$），满足 $m(l, r) + M(l, r)$ 为奇数；
- 将整个段 $a_l, a_{l+1}, \dots, a_r$ 翻转。换句话说，对于所有满足 $l \le i \le r$ 的 $i$，同时执行 $a_i := a_{l+r-i}$。

请判断你是否能通过一系列操作使数组 $a$ 变为非递减数组。

**定义**：数组 $[b_1, b_2, \dots, b_k]$ 被称为非递减的，当且仅当 $b_1 \le b_2 \le \dots \le b_k$。

## 输入格式

每个测试包含多个测试用例。第一行包含一个整数 $t$ ($1 \le t \le 10^4$)，表示测试用例的数量。接下来是测试用例的描述。

每个测试用例的第一行包含一个整数 $n$ ($1 \le n \le 2 \cdot 10^5$) — 数组 $a$ 的长度。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \dots, a_n$ ($1 \le a_i \le n$) — 数组的元素。

保证所有测试用例的 $n$ 之和不超过 $2 \cdot 10^5$。

## 输出格式

对于每个测试用例，如果你能使数组 $a$ 变为非递减的，则输出 `YES`，否则输出 `NO`。

你可以以任意大小写形式输出答案。例如，字符串 `yEs`、`yes`、`Yes` 和 `YES` 都将被识别为肯定答复。

## 示例

**输入**

```
6
4
1 1 2 3
3
2 1 3
5
5 4 3 2 1
6
4 1 2 3 3 6
5
4 2 4 2 4
6
3 3 1 5 5 2
```

**输出**

```
YES
YES
NO
YES
NO
NO
```

## 说明

在第一个测试用例中，数组已经是非递减的，因此输出 `YES`。

在第二个测试用例中，选择 $l=1, r=2$。我们有 $\min(2,1) + \max(2,1) = 2+1=3$，为奇数。因此，对所有 $1 \le i \le 2$ 同时执行 $a_i := a_{3-i}$ 后，得到 $a = [1, 2, 3]$，是非递减的。

考虑第四个测试用例：

- 操作 1：选择 $l=1, r=3$。数组变为 $a = [2, 1, 4, 3, 3, 6]$。
- 操作 2：选择 $l=3, r=5$。数组变为 $a = [2, 1, 3, 3, 4, 6]$。
- 操作 3：选择 $l=1, r=2$。数组变为 $a = [1, 2, 3, 3, 4, 6]$。

在第五个测试用例中，注意无法选择满足条件的下标 $l$ 和 $r$，因此输出 `NO`。
