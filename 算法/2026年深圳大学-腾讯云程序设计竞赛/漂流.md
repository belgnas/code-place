# **【算法题】漂流**

**时间限制**: C/C++/Rust/Pascal 6秒，其他语言12秒  
**空间限制**: C/C++/Rust/Pascal 512 M，其他语言1024 M  
**Special Judge, 64bit IO Format**: %lld

## 题目描述

![图1](https://uploadfiles.nowcoder.com/images/20260408/0_1775611680963/3304ABDD147D8E140B2CEF3201BD8372)

长良、希、瑞穗、拉吉塔尼 … 三十六名少男少女，漂流在无尽的虚空与荒诞的规则之中。时间在这里失去了意义，每一次跃迁的背后，都是一个拥有独立法则的全新维度。

拉吉塔尼习惯用数学来解析这个荒诞的宇宙。他发现，每个世界都有一个固定的「坐标」—— 一个正整数 $n$。而世界的基础构造，如同剥洋葱一般，可以被唯一地分解为若干个「质数维度」的乘积。

由算术基本定理，任意正整数 $n$ 都可以唯一地分解为素数幂的乘积：  
$$ n = p_1^{a_1} \times p_2^{a_2} \times \cdots \times p_k^{a_k} $$  
其中 $p_i$ 是当前坐标包含的、严格递增的素数序列（即 $p_1 < p_2 < \cdots < p_k$），且涉入深度 $a_i > 0$。

在这个多维宇宙中，只有当高维度的涉入深度不超过低维度时，世界的法则才不会坍塌。拉吉塔尼将这种能够维持稳定的坐标称为 **漂流数**。

具体而言，定义一个 **漂流数** 为满足以下条件的正整数：对于任意的 $i < j$，都有 $a_i \ge a_j$。即该数在唯一分解后，质数因子对应的指数是单调不增的。

拉吉塔尼在随身携带的笔记本上写下了几个例子：
- $1$ 是一片纯粹的虚无，它被视为满足条件的漂流数。
- $2 = 2^1$，是漂流数。
- $18 = 2^1 \times 3^2$ 不是漂流数，因为对于 $p_1 = 2, p_2 = 3$，其指数 $a_1 = 1 < a_2 = 2$。
- $360 = 2^3 \times 3^2 \times 5^1$ 是漂流数，因为它的指数序列满足 $3 \ge 2 \ge 1$。

前方是深不见底的漆黑，而希眼中的光芒指向了坐标轴的远方。长良需要知道，在坐标范围不超过 $n$ 的所有世界中，有多少个世界是稳定的「漂流世界」。

定义 $F(n)$ 为不超过 $n$ 的漂流数的个数。给定当前视野的边界 $n$，请协助长良求出 $F(n)$。

## 输入描述

输入包含一个正整数 $n$ ($1 \le n \le 10^{12}$)，表示希眼里的光所能观测到的最远坐标边界。

## 输出描述

输出一个整数，表示 $F(n)$，即不超过 $n$ 的漂流数的个数。

## 示例 1

**输入**

```
100
```

**输出**

```
94
```

**说明**

当 $n = 100$ 时，$F(100) = 94$。  
在 $1$ 到 $100$ 的所有正整数中，仅有以下 $6$ 个数不符合「漂流数」的定义（即在其唯一分解式 $n = p_1^{a_1} p_2^{a_2} \dots p_k^{a_k}$ 中，存在 $i < j$ 使得 $a_i < a_j$）：

- $18 = 2^1 \times 3^2$：质数序列为 $2, 3$，指数序列为 $(1, 2)$。由于 $1 < 2$，不满足单调不增。
- $50 = 2^1 \times 5^2$：质数序列为 $2, 5$，指数序列为 $(1, 2)$。由于 $1 < 2$，不满足单调不增。
- $54 = 2^1 \times 3^3$：质数序列为 $2, 3$，指数序列为 $(1, 3)$。由于 $1 < 3$，不满足单调不增。
- $75 = 3^1 \times 5^2$：质数序列为 $3, 5$，指数序列为 $(1, 2)$。由于 $1 < 2$，不满足单调不增。
- $90 = 2^1 \times 3^2 \times 5^1$：质数序列为 $2, 3, 5$，指数序列为 $(1, 2, 1)$。由于 $a_1 = 1 < a_2 = 2$，不满足单调不增。
- $98 = 2^1 \times 7^2$：质数序列为 $2, 7$，指数序列为 $(1, 2)$。由于 $1 < 2$，不满足单调不增。

其余的 $100 - 6 = 94$ 个数均满足漂流数的定义。
