# **【算法题】A. Koshary**

**时间限制**: 1秒  
**内存限制**: 256 MB

## 题目描述

尤瑟夫位于坐标 $(0,0)$，想要到达位于 $(x,y)$ 的一盘库夏里。

为了到达那里，尤瑟夫迈出**长步**。从任意点 $(a,b)$，一个长步可以移动到：

- $(a + 2, b)$ 或 $(a, b + 2)$

然而，尤瑟夫在整个行程中最多只能迈出**一次短步**。一个短步可以移动到：

- $(a + 1, b)$ 或 $(a, b + 1)$

尤瑟夫能否正好到达库夏里盘子的确切坐标 $(x,y)$？

## 输入格式

第一行包含一个整数 $t$ ($1 \le t \le 100$) — 测试用例的数量。

每个测试用例包含两个整数 $x$ 和 $y$ ($1 \le x, y \le 10$) — 库夏里盘子的坐标。

## 输出格式

对于每个测试用例，如果尤瑟夫可以到达库夏里盘子，则输出 `YES`，否则输出 `NO`。

你可以以任意大小写形式输出答案。例如，字符串 `yEs`、`yes`、`Yes` 和 `YES` 都将被识别为肯定答复。

## 示例

**输入**

```
6
1 1
1 2
4 6
5 9
7 2
10 10
```

**输出**

```
NO
YES
YES
NO
YES
YES
```

## 说明

在第一个测试用例中，无法使用上述步伐到达 $(1,1)$。

在第二个测试用例中，一个最优的步骤序列为：
$(0, 0) \xrightarrow{\texttt{long}} (0, 2) \xrightarrow{\texttt{short}} (1, 2)$

在第三个测试用例中，一个最优的步骤序列为：
$(0, 0) \xrightarrow{\texttt{long}} (0, 2) \xrightarrow{\texttt{long}} (2, 2) \xrightarrow{\texttt{long}} (2, 4) \xrightarrow{\texttt{long}} (4, 4) \xrightarrow{\texttt{long}} (4, 6)$
