# **【算法题】B. Deletion Sort（删除排序）**
每测试用例时间限制：1 秒
每测试用例内存限制：256 兆字节

### 题目描述
AksLolCoding正在对一个由 \(n\) 个正整数组成的数组 \(a\) 玩一个游戏。在每一轮中：
- 如果数组 \(a\) 是非递减的*，游戏结束。
- 否则，AksLolCoding可以选择任意一个元素并将其从数组中移除。

确定游戏结束后数组中可能剩余的最少元素数量。

### 名词解释
数组 \(a\) 是非递减的，当且仅当对于所有 \(1 \leq i \leq m - 1\)，都有 \(a_i \leq a_{i + 1}\)，其中 \(m\) 是数组 \(a\) 的长度。

### 输入
第一行包含一个整数 \(t\)（\(1 \leq t \leq 1000\)），表示测试用例的数量。

每个测试用例的第一行包含一个整数 \(n\)（\(1 \leq n \leq 10\)）。

每个测试用例的第二行包含 \(n\) 个整数，即数组 \(a\) 的元素（\(1 \leq a_i \leq 100\)）。

### 输出
对于每个测试用例，输出一个整数：数组排序后可能剩余的最少元素数量。

### 示例
- **输入**
```
3
4
1 4 2 3
1
100
2
6 7
```
- **输出**
```
1
1
2
```

### 说明
在第一个测试用例中，通过按顺序移除 \(1\)、\(2\) 和 \(3\)，可以实现最少剩余 \(1\) 个元素。

在第二个和第三个测试用例中，无法移除任何元素。
