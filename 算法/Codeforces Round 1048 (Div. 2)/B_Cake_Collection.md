# **【算法题】B. Cake Collection（蛋糕收集）**
- **每个测试的时间限制**：1 秒
- **每个测试的内存限制**：512 兆字节

梅布尔想为巧克拉和香草烤制一些蛋糕。

一天，她发现了 $n$ 个神奇的蛋糕烤箱。第 $i$ 个烤箱每秒烤制 $a_i$ 个蛋糕。蛋糕会留在各自的烤箱中，直到被收集。

在每秒结束时，她可以传送到任意一个烤箱（包括她当前所在的烤箱），并收集截至该时刻在那个烤箱中积累的所有蛋糕。

你的任务是确定梅布尔在 $m$ 秒内最多能收集多少个蛋糕。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 $t$（$1 \leq t \leq 1000$）。接下来是测试用例的描述。

每个测试用例的第一行包含两个整数 $n$ 和 $m$（$1 \leq n \leq 10^5$，$1 \leq m \leq 10^8$）—— 神奇烤箱的数量以及梅布尔收集蛋糕的总秒数。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \leq a_i \leq 10^5$）—— 第 $i$ 个烤箱每秒烤制的蛋糕数量。

保证所有测试用例中 $n$ 的总和不超过 $2 \cdot 10^5$。

### 输出
对于每个测试用例，输出一个整数，表示梅布尔在 $m$ 秒内最多能收集的蛋糕数量。

### 示例
**输入**
```
3
3 4
1 2 3
3 2
1 2 3
1 1000
100000
```
**输出**
```
20
8
100000000
```

### 注意
对于第一个测试用例，一种最优方案如下：
 - 在第一秒结束时，烤箱中分别有 $1$、$2$ 和 $3$ 个蛋糕。梅布尔传送到烤箱 $3$ 并收集所有 $3$ 个蛋糕。烤箱 $3$ 现在剩余 $0$ 个蛋糕。
 - 在第二秒结束时，烤箱中分别有 $2$、$4$ 和 $3$ 个蛋糕。梅布尔传送到烤箱 $1$ 并收集所有 $2$ 个蛋糕。烤箱 $1$ 现在剩余 $0$ 个蛋糕。
 - 在第三秒结束时，烤箱中分别有 $1$、$6$ 和 $6$ 个蛋糕。梅布尔传送到烤箱 $2$ 并收集所有 $6$ 个蛋糕。烤箱 $2$ 现在剩余 $0$ 个蛋糕。
 - 在第四秒结束时，烤箱中分别有 $2$、$2$ 和 $9$ 个蛋糕。梅布尔传送到烤箱 $3$ 并收集所有 $9$ 个蛋糕。烤箱 $3$ 现在剩余 $0$ 个蛋糕。
  梅布尔总共收集了 $3 + 2 + 6 + 9 = 20$ 个蛋糕。

对于第二个测试用例，一种最优方案如下：
 - 在第一秒结束时，烤箱中分别有 $1$、$2$ 和 $3$ 个蛋糕。梅布尔传送到烤箱 $2$ 并收集所有 $2$ 个蛋糕。烤箱 $2$ 现在剩余 $0$ 个蛋糕。
 - 在第二秒结束时，烤箱中分别有 $2$、$2$ 和 $6$ 个蛋糕。梅布尔传送到烤箱 $3$ 并收集所有 $6$ 个蛋糕。烤箱 $3$ 现在剩余 $0$ 个蛋糕。
  梅布尔总共收集了 $2 + 6 = 8$ 个蛋糕。
