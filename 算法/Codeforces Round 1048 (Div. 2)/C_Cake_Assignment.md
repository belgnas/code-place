# **【算法题】C. Cake Assignment（蛋糕分配）**
- **每个测试的时间限制**：2 秒
- **每个测试的内存限制**：256 兆字节

巧克力和香子兰都很喜欢蛋糕。今天，蛋糕店老板总共给了她们 $2^{k + 1}$ 个蛋糕。这些蛋糕被平均分配，所以她们最初每人收到 $2^k$ 个蛋糕。

然而，巧克力和香子兰现在想要重新分配蛋糕，使得巧克力最终恰好得到 $x$ 个蛋糕，而巧克力与香子兰得到剩下的 $2^{k + 1} - x$ 个蛋糕。

在一步操作中，她们可以确切地执行以下两个操作之一：
1. 巧克力把她一半的蛋糕给香子兰。仅当巧克力当前拥有偶数个蛋糕时，此操作才被允许。
2. 香子兰把她一半的蛋糕给巧克力。仅当巧克力与香子兰当前拥有偶数个蛋糕时，此操作才被允许。

你的任务是确定达到目标分配所需的最少步骤数，并输出任何能实现该最少步骤的有效操作序列。

可以证明，在给定的约束条件下，总是存在有效的解决方案，并且所需的最少步骤数最多为 120 步。

## 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 $t$（$1 \leq t \leq 1000$）。接下来是测试用例的描述。

每个测试用例的第一行包含两个整数 $k$ 和 $x$（$1 \leq k \leq 60$，$1 \leq x \leq 2^{k + 1} - 1$）—— 最初每人收到 $2^k$ 个蛋糕，$x$ 是重新分配后巧克力应拥有的蛋糕数。

## 输出
对于每个测试用例，输出一个整数 $n$（$0 \leq n \leq 120$），表示她们相应地重新分配蛋糕所需的最少步骤数。

在下一行，输出 $n$ 个整数 $o_1, o_2, \ldots, o_n$（$o_i = 1$ 或 $o_i = 2$），其中 $o_i = 1$ 表示在第 $i$ 步中，巧克力把她一半的蛋糕给香子兰（操作 1），$o_i = 2$ 表示巧克力与香子兰把她一半的蛋糕给巧克力（操作 2）。

## 示例
**输入**
```
4
2 3
2 4
3 7
2 5
```
**输出**
```
2
2 1 
0

3
2 2 1 
2
1 2 
```

## 注意
在第一个测试用例中，她们可以使用以下步骤，使巧克力恰好有 $x = 3$ 个蛋糕，香子兰恰好有 $2^{k + 1} - x = 5$ 个蛋糕。我们用 $\{a, b\}$ 表示巧克力当前有 $a$ 个蛋糕，香子兰当前有 $b$ 个蛋糕。

$\{4, 4\} \stackrel{o_1 = 2}{\longrightarrow} \{6, 2\} \stackrel{o_2 = 1}{\longrightarrow} \{3, 5\}$

在第二个测试用例中，巧克力已经恰好有 $x = 4$ 个蛋糕，香子兰也已经恰好有 $2^{k + 1} - x = 4$ 个蛋糕，所以不需要任何操作。

在第三个测试用例中，她们可以使用以下步骤，使巧克力恰好有 $x = 7$ 个蛋糕，香子兰恰好有 $2^{k + 1} - x = 9$ 个蛋糕。

$\{8, 8\} \stackrel{o_1 = 2}{\longrightarrow} \{12, 4\} \stackrel{o_2 = 2}{\longrightarrow} \{14, 2\} \stackrel{o_3 = 1}{\longrightarrow} \{7, 9\}$
