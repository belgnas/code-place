# **【算法题】C. Fibonacci Sum（斐波那契和）**
每测试用例时间限制：1 秒
每测试用例内存限制：512 兆字节

### 题目描述
设 \(f(i)\) 表示著名的斐波那契数列中的第 \(i\) 个数。正式定义如下：
\[ f(i)=
\begin{cases}
1 & \text{如果 } i \leq 2 \\
f(i - 1) + f(i - 2) & \text{如果 } i > 2
\end{cases}
\]

设 \(g(x)\) 表示数字 \(x\) 的二进制表示中 \(1\) 的个数。例如，\(g(5)=g(101_{(2)}) = 2\)，\(g(15)=g(1111_{(2)}) = 4\) 。

你的任务是计算以下值：

\(\sum_{i = 1}^{n} f(g(i)) \bmod (10^9 + 7)\)

### 输入
输入包含一个仅由 '0' 或 '1' 组成的字符串 \(s\)（\(1 \leq |s| \leq 10^7\)），表示二进制形式的数字 \(n\) 。

保证输入不包含前导 \(0\) 。

### 输出
输出一个整数，表示对 \((10^9 + 7)\) 取模后的答案。

### 示例
- **输入**
```
1
```
- **输出**
```
1
```
- **输入**
```
10
```
- **输出**
```
2
```
- **输入**
```
11
```
- **输出**
```
3
```

### 说明
可以计算得出 \(f(g(1_{(2)})) = 1\)，\(f(g(10_{(2)})) = 1\)，\(f(g(11_{(2)})) = 1\) 。
