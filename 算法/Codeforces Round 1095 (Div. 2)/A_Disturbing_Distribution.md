# **【算法题】A. Disturbing Distribution**

**时间限制**: 1秒  
**内存限制**: 256 MB

## 题目描述

你有一个数组 $[a_1, a_2, \dots, a_n]$。你希望通过执行以下操作任意次来将数组清空：

- 选择一个下标序列 $1 \le i_1 < i_2 < \dots < i_k \le |a|$（注意 $|a|$ 表示数组 $a$ 的当前长度），满足 $a_{i_1} \le a_{i_2} \le \dots \le a_{i_k}$。
- 从数组 $a$ 中删除元素 $a_{i_1}, a_{i_2}, \dots, a_{i_k}$。

此操作的代价为 $a_{i_1} \times a_{i_2} \times \cdots \times a_{i_k}$。

请确定将数组 $a$ 中所有元素移除所需的最小总代价。总代价等于所有执行操作的代价之和。

由于答案可能非常大，请输出答案对 $676767677$ 取模的结果。

## 输入格式

每个测试包含多个测试用例。第一行包含一个整数 $t$ ($1 \le t \le 500$)，表示测试用例的数量。接下来是测试用例的描述。

每个测试用例的第一行包含一个整数 $n$ ($1 \le n \le 100$) — 数组 $a$ 的长度。

每个测试用例的第二行包含 $n$ 个整数 $a_1, a_2, \dots, a_n$ ($1 \le a_i \le 100$) — 数组中的元素。

## 输出格式

对于每个测试用例，输出一个整数 — 将数组 $a$ 清空所需的最小总代价。由于答案可能很大，请输出答案对 $676767677$ 取模的结果。

## 示例

**输入**

```
3
5
1 2 1 2 3
3
3 2 1
4
1 1 1 1
```

**输出**

```
7
6
1
```

## 说明

对于第一个测试用例：

- 操作 1: 选择 $i_1 = 1$, $i_2 = 2$, $i_3 = 4$。代价为 $1 \cdot 2 \cdot 2 = 4$。删除这些下标对应的元素后，数组变为 $[1, 3]$。
- 操作 2: 选择 $i_1 = 1$ 和 $i_2 = 2$。代价为 $1 \cdot 3 = 3$。删除这些元素后，数组变为空。

总代价为 $4 + 3 = 7$。可以证明这是可能的最小总代价。
