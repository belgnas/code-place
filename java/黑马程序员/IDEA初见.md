# IDEA  

## IDEA项目结构介绍  
project（项目）-> module（模块）-> package（包）-> class（类）

## IDEA常用设置
1. 提示忽略大小写
2. 修改注释颜色
3. 修改字体
4. 自动导包
5. 设置背景图片


## IDEA的项目与模块操作
1. 类的操作
   1. 新建类
   2. 删除类（右键文件修改）
   3. 修改类名（不要直接在代码中修改，因为文件名和类名要保持一致，可以对文件右键，在弹出的选项中选择重命名（rename），或者直接快捷键 Shift + F6）
2. 模块的操作
   1. 新建模块（File -> Project Structure -> Modules -> +号 -> New Module -> 起名字 -> 选择yes、finish等保存）
   2. 删除模块（后续讲）
   3. 修改模块（后续讲）
   4. 导入模块（后续讲）
3. 项目的操作
   1. 关闭项目（Flie -> Close File）
   2. 新建项目（File -> New File）
   3. 打开项目（后续讲）（鼠标双击）
   4. 修改项目（后续讲）
