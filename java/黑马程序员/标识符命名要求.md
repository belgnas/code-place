# 标识符

## 硬性要求
1. 标识符可以由字母、数字、下划线(_)、美元符号($)组成
2. 标识符不能以数字开头
3. 标识符不能是关键字
4. 标识符不能是空格
5. 区分大小写

## 软性建议
1. 方法、变量（小驼峰命名法）
    1. 当标识符为一个单词时，全部小写  
        例：name
    2. 当标识符为多个单词时，第一个单词首字母小写，其余每个单词首字母大写  
        例：firstName
2. 类（大驼峰命名法）
    1. 当标识符为一个单词时，首字母大写  
        例：Student
    2. 当标识符为多个单词时，每个单词首字母大写  
        例：StudentName
