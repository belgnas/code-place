# **【算法题】Get The Number（得到数字）**

时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
空间限制：C/C++/Rust/Pascal 256 M，其他语言512 M
Special Judge, 64bit IO Format: %lld

### 题目描述 
小白月赛起步早但频率低，牛客周赛起步晚但频率高。

终于，在2026年1月中下旬的周末，两者产生了交集，Round127真是个神秘的场次呢...

小苯有三个正整数 \(A\)、\(B\)、\(C\)，他可以使用恰好一次四则运算（加法、减法、乘法、除法）将 \(A\) 和 \(B\) 组合起来，得到一个新的整数。
具体来说，四种操作如下：
1. \(A + B\)
2. \(A - B\)
3. \(A × B\)
4. \(A ÷ B\)（仅当 \(A\) 能被 \(B\) 整除时，注意：除法必须保证能整除，且结果仍为整数。）

你的任务就是判断 \(C\) 是否等于上面四个算式之一的结果。

### 输入描述:
一行三个正整数 \(A\)、\(B\)、\(C\) \((1 \leq A, B, C \leq 10)\)。

### 输出描述:
如果 \(C\) 可以通过一次四则运算得到，输出一行 "YES"，否则输出一行 "NO"。

### 示例1
- **输入**
```
3 5 8
```
- **输出**
```
YES
```
- **说明**
第一组数据：\(3 + 5 = 8\)，符合。

### 示例2
- **输入**
```
10 2 5
```
- **输出**
```
YES
```
- **说明**
第二组数据：\(10 ÷ 2 = 5\)，符合。

### 示例3
- **输入**
```
10 2 6
```
- **输出**
```
NO
```
