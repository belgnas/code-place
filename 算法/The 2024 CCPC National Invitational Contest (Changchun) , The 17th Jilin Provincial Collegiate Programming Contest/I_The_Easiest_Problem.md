# **【算法题】I. The Easiest Problem（最简单的问题）**
每测试用例时间限制：1 秒
每测试用例内存限制：512 兆字节

### 题目描述
上课时，老师经常说“现在扫码签到。”

现在，格夫想问你这句话里有多少个小写英文字母。

### 输入
输入仅包含一行，内容为句子“Scan the QR code to sign in now.”

### 输出
输出一个整数，表示问题的答案。

### 示例
- **输入**
```
Scan the QR code to sign in now.
```
- **输出**
```
-1
```

### 说明
示例中的输出是不正确的。需要你自己找出答案，并获得正确的判定结果。
