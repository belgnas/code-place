# **【算法题】绫地宁宁的起爆器**

**时间限制**: C/C++/Rust/Pascal 1秒，其他语言2秒  
**空间限制**: C/C++/Rust/Pascal 256 M，其他语言512 M  
**Special Judge, 64bit IO Format**: %lld

## 题目描述

![图1：辅助示意图](https://uploadfiles.nowcoder.com/images/20260408/0_1775613051263/D2B5CA33BD970F64A6301FA75AE2EB22)

绫地宁宁有一个**起爆器**（没有引申义），其状态可用一个长度为 $n$ 的字符串 $s$ 表示。起爆器中包含了 $m$ 个**组件**，第 $i$ 个组件对应字符串 $s$ 的一个非空子串 $t_i = s[l_i, r_i]$。组件 $t_i$ 的**能量**被定义为它在 $s$ 中的出现次数。起爆器的能量被定义为 $m$ 个组件的能量之和。

绫地宁宁邀请保科进行游戏：绫地宁宁先手，双方轮流进行操作。在每回合中，当前选手必须选择某个一个组件 $t_i$，执行以下两个操作中的至少一个：

- 选择任意一个字符 $c$，令 $t_i \leftarrow c + t_i$。
- 选择任意一个字符 $c$，令 $t_i \leftarrow t_i + c$。

由于起爆器非常不稳定，操作后必须保证起爆器的能量不发生变化。无法进行合法操作的选手判负，请判断最终获胜的选手。可以证明游戏不存在平局情况，即游戏可以在有限回合内判定胜负。

## 输入描述

第 $1$ 行输入两个整数 $n,m(1 \le n,m \le 2 \cdot 10^5)$，分别表示字符串 $s$ 的长度以及起爆器的组件数量。第 $2$ 行输入一个长度为 $n$ 的起爆器字符串 $s$。

接下来 $m$ 行，每行输入 $2$ 个整数 $l_i, r_i(1 \le l_i \le r_i \le n)$。表示组件 $t_i = s[l_i, r_i]$。

## 输出描述

输出 $1$ 行，如果绫地宁宁获胜，输出 `0d00`。如果保科获胜，输出 `bkzs`。

## 示例 1

**输入**

```
4 1
aaaa
3 4
```

**输出**

```
bkzs
```

## 示例 2

**输入**

```
9 1
abbbabbba
6 8
```

**输出**

```
0d00
```
