# **【算法题】D. Palindromex**

**时间限制**: 2秒  
**内存限制**: 256 MB

## 题目描述

尤瑟夫给了你一个长度为 $2n$ 的数组 $a$。每个整数 $x \in [0, n-1]$ 在数组中恰好出现两次。

你的任务是找到一个子数组 $a_l, a_{l+1}, \dots, a_r$，该子数组是一个**回文数组**，且其 **$\operatorname{mex}(a_l, a_{l+1}, \dots, a_r)$** 最大化。请输出这个可能的最大值。

**定义**：

- **回文**：一个序列，正读和反读都相同。例如，字符串 `"z"`, `"aaa"`, `"aba"`, `"abccba"` 是回文的，但字符串 `"codeforces"`, `"reality"`, `"ab"` 不是。
- **$\operatorname{mex}$（最小排除值）**：一个整数数组中未出现的最小非负整数。例如：
  - $[2,2,1]$ 的 $\operatorname{mex}$ 是 $0$，因为 $0$ 不在数组中。
  - $[3,1,0,1]$ 的 $\operatorname{mex}$ 是 $2$，因为 $0$ 和 $1$ 在数组中，但 $2$ 不在。
  - $[0,3,1,2]$ 的 $\operatorname{mex}$ 是 $4$，因为 $0,1,2,3$ 都在数组中，但 $4$ 不在。

## 输入格式

第一行包含一个整数 $t$ ($1 \le t \le 10^4$) — 测试用例的数量。

每个测试用例的第一行包含一个整数 $n$ ($1 \le n \le 10^5$)。

每个测试用例的第二行包含 $2n$ 个整数 $a_1, a_2, \dots, a_{2n}$ ($0 \le a_i \le n-1$)。保证范围 $[0, n-1]$ 内的每个整数恰好出现两次。

保证所有测试用例的 $2n$ 之和不超过 $2 \cdot 10^5$。

## 输出格式

对于每个测试用例，输出一个整数 — 任意回文子数组的最大 $\operatorname{mex}$ 值。

## 示例

**输入**

```
6
4
1 2 0 3 3 0 2 1
2
0 1 0 1
2
1 1 0 0
3
2 0 2 1 1 0
4
0 1 3 0 3 1 2 2
3
0 1 2 1 0 2
```

**输出**

```
4
2
1
1
2
3
```

## 说明

在第一个测试用例中，唯一的最优选择是子数组 $a[1,8] = [1,2,0,3,3,0,2,1]$，它是回文的，且 $\operatorname{mex}$ 为 $4$。

在第二个测试用例中，可以选择的最优子数组之一是 $a[2,4] = [1,0,1]$，它是回文的，且 $\operatorname{mex}$ 为 $2$。

在第三个测试用例中，我们可以选择 $a[3,3] = [0]$，它是回文的，且 $\operatorname{mex}$ 为 $1$。没有其他回文子数组的 $\operatorname{mex}$ 大于 $1$。
