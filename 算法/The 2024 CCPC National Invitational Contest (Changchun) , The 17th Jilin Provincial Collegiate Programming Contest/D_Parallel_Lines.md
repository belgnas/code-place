# **【算法题】D. Parallel Lines（平行线）**
每测试用例时间限制：3 秒
每测试用例内存限制：512 兆字节

### 题目描述
从前，在二维平面上有 \(k\) 条平行线和 \(n\) 个点在这些线上。已知每条线上至少有两个点。

现在给定这 \(n\) 个点，你的任务是找出这 \(k\) 条平行线。

### 输入
第一行包含两个整数 \(n\)，\(k\)（\(2 \leq n \leq 10^4\)，\(1 \leq k \leq \min(50, \frac{n}{2})\)），分别表示点的数量和平行线的数量。

接下来 \(n\) 行，第 \(i\) 行包含两个整数 \(x_i\)，\(y_i\)（\(1 \leq x_i, y_i \leq 10^9\)），表示第 \(i\) 个点的坐标。

保证 \(n\) 个点两两不同（即对于所有 \(1 \leq i < j \leq n\)，要么 \(x_i \neq x_j\)，要么 \(y_i \neq y_j\) 成立）。

### 输出
输出包含 \(k\) 行。

在第 \(i\) 行，首先输出一个整数 \(m_i\)（\(2 \leq m_i \leq n\)），表示第 \(i\) 条平行线上点的数量。然后输出 \(m_i\) 个整数 \(x_1, x_2, \cdots, x_{m_i}\)，表示第 \(i\) 条线上点的索引。

你的输出应满足每个点出现且仅出现在一条线上，并且 \(k\) 条线相互平行且不同。

保证存在一种有效的方法将 \(n\) 个点分配到 \(k\) 条平行线上。如果存在多个解，输出任意一个即可。

### 示例
- **输入**
```
4 2
1 3
2 5
4 7
5 9
```
- **输出**
```
2 3 4 
2 1 2
```
