# **【算法题】A. The 67th Integer Problem（第67个整数问题）**
每测试用例时间限制：1 秒
每测试用例内存限制：256 兆字节

### 题目描述
欢迎来到新世界，尊贵的地球访客。你被猕猴召唤而来，这是一只拥有四条腿、自命不凡且对“小菜一碟”这个词极度上瘾的灵长类动物。你即将开启一段极其重要的旅程，重要到令人难以置信。没有任何旅程会比这更重要（而且这个叙述者说的话再也不会听起来如此特别地…… 充满橙色的感觉）。恳请你与猕猴合作，因为他的愤怒（和无业状态）无穷无尽。容不得任何错误或无能。表现糟糕将受到法律的严惩。

猕猴得到一个整数 \(x\) 。你的任务是选择一个整数 \(y\) ，使得 \(\min(x, y)\) \(^{*}\) 的值最大化。

如果有多个有效的 \(y\) ，你可以输出其中任何一个。

### 名词解释
\(\min(x, y)\) 定义为整数 \(x\) 和 \(y\) 中的最小值。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 6767\)）。接下来是测试用例的描述。

每个测试用例的唯一一行包含一个整数 \(x\)（\(-67 \leq x \leq 67\)）。

### 输出
对于每个测试用例，输出一个整数 \(y\)（\(-67 \leq y \leq 67\)），使得 \(\min(x, y)\) 最大化。

### 示例
- **输入**
```
3
1
3
5
```
- **输出**
```
2
4
6
```

### 说明
在第一个测试用例中，\(2\) 是一个可能的答案，因为 \(\min(1, 2) = 1\) ，可以证明这是最大值。
