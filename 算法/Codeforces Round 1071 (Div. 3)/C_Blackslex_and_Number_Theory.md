# **【算法题】C. Blackslex and Number Theory（布莱克斯莱克斯与数论）**

每个测试的时间限制：2 秒
每个测试的内存限制：256 兆字节

布莱克斯莱克斯学习太过努力，以至于做梦都梦到数字。来解决他梦中出现的下面这个问题。

给定一个数组\(a_1, a_2, \cdots, a_n\) 。

在一次操作中，你选择一个索引\(i\)（\(1 \leq i \leq n\)）和一个至少为\(k\)的整数\(x\)，并设置\(a_i := a_i \bmod x\)，其中\(u \bmod v\)表示\(u\)除以\(v\)的余数。

你的目标是使数组的所有元素相同。在所有正整数\(k\)中，确定最大的\(k\)，使得存在上述操作的有限序列，能使数组所有元素相等。

### 输入
第一行包含一个整数\(t\)（\(1 \leq t \leq 10^4\)）——测试用例的数量。

每个测试用例的第一行包含一个整数\(n\)（\(2 \leq n \leq 2 \cdot 10^5\)）。

第二行包含\(n\)个整数\(a_1, a_2, \cdots, a_n\)（\(1 \leq a_i \leq 10^9\)，\(a\)中的所有值都不同）。

保证所有测试用例中\(n\)的总和不超过\(2 \cdot 10^5\)。

### 输出
对于每个测试用例，打印一个整数 —— 最大的正整数\(k\)，使得使用模\(x\)且\(k \leq x\)的任意次数操作，都有可能使数组的所有元素相同。

### 示例
- **输入**
```
3
3
5 7 9
2
2 3
7
11 74 5 22 52 97 82
```
- **输出**
```
5
2
6
```
