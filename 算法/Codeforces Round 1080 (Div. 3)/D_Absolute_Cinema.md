# **【算法题】D. Absolute Cinema（绝对值运算谜题）**
每测试用例时间限制：2 秒
每测试用例内存限制：256 兆字节

### 题目描述
存在一个由 \(n\) 个整数组成的隐藏序列 \(a_1, a_2, \cdots, a_n\)（\(n \geq 2\)）。保证对于所有 \(1 \leq i \leq n\)，都有 \(|a_i| \leq 1000\) 。

我们定义一个函数 \(f(x)\) 如下：
\(f(x)=\sum_{i = 1}^{n}a_i \cdot |i - x|\)

给定 \(n\) 个值 \(f(1), f(2), \cdots, f(n)\)，请确定 \(a_1, a_2, \cdots, a_n\) 的值。

保证 \(a_1, a_2, \cdots, a_n\) 的值可以唯一确定。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 10^4\)）。接下来是测试用例的描述。

每个测试用例的第一行包含一个整数 \(n\)（\(2 \leq n \leq 300000\)）。

每个测试用例的第二行包含 \(n\) 个整数 \(f(1), f(2), \cdots, f(n)\)（\(-10^{14} \leq f(i) \leq 10^{14}\)）。

保证所有测试用例的 \(n\) 之和不超过 \(300000\) 。

### 输出
对于每个测试用例，在单独一行输出 \(n\) 个整数 \(a_1, a_2, \cdots, a_n\)（\(|a_i| \leq 1000\)）。

保证 \(a_1, a_2, \cdots, a_n\) 的值可以唯一确定。

### 示例
- **输入**
```
4
4
17 9 9 13
6
-37 -32 -15 4 27 42
5
-26 -32 -24 -4 2
2
420 -69
```
- **输出**
```
1 4 2 3
3 6 1 2 -4 -7
-6 7 6 -7 -6
-69 420
```

### 说明
在第一个测试用例中，隐藏序列是 \(a = [1, 4, 2, 3]\) 。

\(f(1), f(2), \cdots, f(n)\) 的值如下：
\(f(1)=1 \cdot |1 - 1| + 4 \cdot |2 - 1| + 2 \cdot |3 - 1| + 3 \cdot |4 - 1| = 0 + 4 + 4 + 9 = 17\)；
\(f(2)=1 \cdot |1 - 2| + 4 \cdot |2 - 2| + 2 \cdot |3 - 2| + 3 \cdot |4 - 2| = 1 + 0 + 2 + 6 = 9\)；
\(f(3)=1 \cdot |1 - 3| + 4 \cdot |2 - 3| + 2 \cdot |3 - 3| + 3 \cdot |4 - 3| = 2 + 4 + 0 + 3 = 9\)；
\(f(4)=1 \cdot |1 - 4| + 4 \cdot |2 - 4| + 2 \cdot |3 - 4| + 3 \cdot |4 - 4| = 3 + 8 + 2 + 0 = 13\)。

# 先记住题目核心公式
题目给的定义：
$$f(x) = a_1|1-x| + a_2|2-x| + \dots + a_n|n-x|$$

代码里的 `vec[i]` **就是题目里的 a[i]**！

---

# 一、为什么中间值要这样算？
## 核心推导（超级简单）
我们看相邻两个 f 值的差：

$$f(x+1) - f(x)$$

这个式子化简后，会得到一个超级干净的结果：
$$f(x+1)-f(x) = (a_1+a_2+\dots+a_x) - (a_{x+1}+\dots+a_n)$$

我们再做一次差分：
$$[f(x+1)-f(x)] - [f(x)-f(x-1)]$$

最终化简结果就是：
$$f(x+1) + f(x-1) - 2f(x) = 2\cdot a_x$$

## 所以：
$$a_x = \frac{f(x+1)+f(x-1)-2f(x)}{2}$$

---

# 这就是代码里这一行的来历！
```cpp
int delta = f[i + 1] + f[i - 1] - 2 * f[i];
vec[i] = delta / 2;
```

### 一句话总结
**中间的 a[i] 可以直接通过 f 数组的二阶差分算出来！**
这是数学推导的固定公式，不是随便写的。

# 二、为什么第一个和最后一个要那样算？
中间的 a[2]...a[n-1] 我们求出来了。
但 **a[1] 和 a[n] 求不出来**，因为它们没有左右两个邻居。

所以我们要**用定义式反推**！

---

## 代入 x = 1 到题目公式里
$$f(1) = a_1\cdot0 + a_2\cdot1 + a_3\cdot2 + \dots + a_n \cdot (n-1)$$

整理一下：
$$f(1) = a_2\cdot1 + a_3\cdot2 + ... + a_{n-1}(n-2) + a_n \cdot (n-1)$$

我们把已知的部分移到左边：
$$f(1) - \sum_{i=2}^{n-1} a_i\cdot (i-1) = a_n \cdot (n-1)$$

左边就是代码里的：
```cpp
calculate(0, f, vec)
```

所以：
$$a_n = \frac{calculate(0)}{n-1}$$

---

## 同理代入 x = n
$$f(n) = a_1(n-1)+a_2(n-2)+...+a_{n-1}\cdot1 + a_n\cdot0$$

移项得：
$$f(n)-\sum_{i=2}^{n-1}a_i\cdot(n-i) = a_1\cdot(n-1)$$

左边就是代码里的：
```cpp
calculate(n-1, f, vec)
```

所以：
$$a_1 = \frac{calculate(n-1)}{n-1}$$
