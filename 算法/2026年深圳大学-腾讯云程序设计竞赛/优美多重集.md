# **【算法题】优美多重集**

**时间限制**: C/C++/Rust/Pascal 2秒，其他语言4秒  
**空间限制**: C/C++/Rust/Pascal 256 M，其他语言512 M  
**Special Judge, 64bit IO Format**: %lld

## 题目描述

一个多重集 $S = \{x_1, x_2, x_3, \dots\}$ 被称为“优美”的，当且仅当 $S$ 同时满足以下三个条件：

- $|S| = n$
- $1 \le x_1 \le x_2 \le \cdots \le x_n \le n$
- $\forall\ i \subseteq [1,n) ,\ x_{i+1} - x_i \le 1$

一个优美多重集的价值定义为 $V(S) = \sum_{i=1}^{n} v_{x_i}$。

求所有不同的优美多重集的价值之和，答案对 $998244353$ 取模。

## 输入描述

每个测试点包含 $T$（$1 \le T \le 10^4$）组数据。

在每组数据中，第一行包含一个整数 $n$（$1 \le n \le 2 \cdot 10^5$）。

接下来一行包含 $n$ 个正整数 $v_1, v_2, \dots, v_n$（$1 \le v_i \le 10^9$）。

保证 $T$ 组数据中的 $n$ 之和不超过 $5 \cdot 10^5$。

## 输出描述

共 $T$ 行，每行输出一个整数，表示所有不同的优美多重集的价值之和，答案对 $998244353$ 取模。

## 示例 1

**输入**

```
3
3
2 2 2
2
1 1
3
1 3 2
```

**输出**

```
48
6
51
```
