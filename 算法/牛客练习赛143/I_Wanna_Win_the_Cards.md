# **【算法题】蓝莓酱和草莓酱的卡牌游戏**

### 题目描述 
蓝莓酱和草莓酱各有 $n$ 张卡牌，蓝莓酱持有的卡牌种族值均为 $1$，而草莓酱持有的卡牌种族值均为 $[2, n + 1]$ 范围内的整数，且草莓酱持有的每张卡牌的种族值均不相同。

两只小可爱都好想要对方手中的卡牌，因此它们决定用一个小游戏决出胜负，规则如下：
 - 双方将所有的卡牌汇总成一堆；
 - 蓝莓酱先手，双方轮流操作；
 - 轮到某方操作时，其必须选择至少 $2$ 张、至多 $k$ 张卡牌，满足所选卡牌之间的种族值两两不同，将这几张卡牌永久移除；
 - 如果该方无法操作，则该方输掉游戏。

在双方都采取最优策略的情况下，哪只小可爱能取得胜利呢？

### 输入描述:
每个测试文件均包含多组测试数据。第一行输入一个整数 $T(1 \leq T \leq 100)$ 代表数据组数，每组测试数据描述如下：

在一行上输入两个整数 $n, k(2 \leq n, k \leq 100)$，表示双方初始各自持有的卡牌数、单轮最高取牌数。

### 输出描述:
对于每一组测试数据，新起一行，如果蓝莓酱获得胜利，输出 “Blueberry”；否则，输出 “Strawberry”。

您可以以任何大小写形式输出答案。例如，字符串 “bLueBerrY”、“blueberry” 都视为蓝莓酱胜利。

### 示例1
**输入**
```
3
2 2
3 2
5 2
```
**输出**
```
Blueberry
Strawberry
Blueberry
```

### 示例2
**输入**
```
3
100 12
100 24
100 48
```
**输出**
```
Blueberry
Strawberry
Blueberry
```
