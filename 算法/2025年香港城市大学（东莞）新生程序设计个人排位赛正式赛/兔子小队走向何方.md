# **【算法题】兔子小队的团结判断**

联邦学生会决定取消兔子小队，兔子小队的成员坚决反对这个决议，于是展开了武力抗议。

然而，这场抗议最终还是失败了，兔子小队只能接受被解散的事实，生活在小兔子公园里勉强度日。

慢慢地，兔子小队逐渐走向了她们的......

结局。

`sensei` 非常关心兔子小队的境遇，经常给她们送去各种物资，并试图帮助她们走出困境。因此，`sensei` 需要经常观察兔子小队各个成员的动向，并判断她们是否还团结在一起。

假设兔子小队有 $n$ 个成员，以公园门口为坐标原点，第 $i$ 个成员所在的位置记为 $a_i$ 。如果所有成员的位置都在坐标原点的同一边，`sensei` 就认为她们仍然紧密团结在一起，否则就认为她们之间可能出现了分歧。

你需要写一个程序，帮助 `sensei` 判断兔子小队的大家是否仍然紧密团结在一起。

### 输入描述:
输入的第一行包含一个正整数 $n$（$1≦n≦10^5$），表示兔子小队的成员人数。

输入的第二行包含 $n$ 个整数：$a_1,a_2,...,a_n$（$1≦|a_i|≦10^9$），第 $i$ 个整数表示第 $i$ 个成员所在的位置。

### 输出描述:
输出一行一个字符串，如果兔子小队的大家仍然紧密团结在一起，输出 `Yes`，否则输出 `No`。

### 示例1
**输入**
```
4
1 3 5 -2
```
**输出**
```
No
```

### 示例2
**输入**
```
6
-1 -3 -2 -9 -5 -1
```
**输出**
```
Yes
```
