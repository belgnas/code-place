# **【算法题】ICPC Balloons（ICPC气球问题）**

时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
空间限制：C/C++/Rust/Pascal 256 M，其他语言512 M
64bit IO Format: %lld

### 题目描述 
在 ICPC 比赛中，正式比赛前通常有热身赛供选手熟悉比赛环境和测试系统。
已知热身赛有A、B、C、D四道题，每道题对应不同颜色的气球：
 - 题目A：红色气球（red）
 - 题目B：橙色气球（orange）
 - 题目C：蓝色气球（blue）
 - 题目D：绿色气球（green）
当选手队伍通过一道题目时，志愿者会根据题目编号发放对应颜色的气球。

现在给定小苯的队伍通过的题目编号（‘A’、‘B’、‘C’ 或 ‘D’），请你输出志愿者应该给他发放的气球颜色。

### 输入描述:
在一行上输入一个大写字母 \(c\)，表示通过的题目编号，保证 \(c \in \{‘A’,‘B’,‘C’,‘D’\}\)。

### 输出描述:
输出一个字符串 \(s(s \in \{‘red’,‘orange’,‘blue’,‘green’\})\)，表示气球颜色。

### 示例1
- **输入**
```
B
```
- **输出**
```
orange
```
- **说明**
根据题目编号与气球颜色的对应关系：
 - 题目 A 对应红色气球 red；
 - 题目 B 对应橙色气球 orange；
 - 题目 C 对应蓝色气球 blue；
 - 题目 D 对应绿色气球 green。
