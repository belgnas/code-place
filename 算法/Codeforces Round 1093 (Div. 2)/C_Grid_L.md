# **【算法题】Grid L（L形网格）**
每个测试用例的时间限制：2 秒
每个测试用例的内存限制：256 兆字节

罗杰有 \(p\) 条单位长度的线段和 \(q\) 个 L 形部件，每个 L 形部件由两条单位长度的线段在直角处连接而成。
![L形部件](https://espresso.codeforces.com/0cbf71a5c455440918b2789177b20f6f676173c2.png)

他想用所有这些部件（没有部件剩余）组成一个 \(n×m\) 的网格。

给定 \(p\) 和 \(q\)，判断是否存在正整数 \(n\) 和 \(m\)，使得恰好使用 \(p\) 条单位长度的线段和 \(q\) 个 L 形部件（可能需要旋转它们）就能组成一个 \(n×m\) 的网格，并给出任意有效的 \(n\) 和 \(m\)。

### 输入
每个测试包含多个测试用例。第一行包含测试用例的数量 \(t\)（\(1 \leq t \leq 100\)）。接下来是测试用例的描述。

每个测试用例的第一行包含两个整数 \(p\) 和 \(q\)（\(1 \leq p, q \leq 10^8\)）。

### 输出
对于每个测试用例，如果存在这样的 \(n\) 和 \(m\)，使得可以用所有部件构造出 \(n×m\) 的网格，则输出任意有效的 \(n\) 和 \(m\)。如果不存在这样的 \(n\) 和 \(m\)，则输出 \(-1\)。

### 示例
**输入**
```
7
1 2
1 3
5 1
2 5
2 10
100000000 100000000
1 1
```
**输出**
```
-1
1 2
1 2
2 2
2 4
-1
-1
```
