# **【算法题】ICPC Time（ICPC 时间问题）**

时间限制：C/C++/Rust/Pascal 1 秒，其他语言 2 秒
空间限制：C/C++/Rust/Pascal 256 M，其他语言 512 M
64bit IO Format: %lld

### 题目描述 
众所周知，一场 ICPC 区域赛的时长是 \(5\) 小时。
现在给定一个整数 \(n(0 \leq n \leq 23)\)，表示区域赛的开始时间（\(n\) 时 \(0\) 分 \(0\) 秒）。
你的任务就是求出：本场区域赛结束时的小时数（注意：按 \(24\) 小时制计时，即从 \(0\) 点开始到 \(23\) 点为一天，答案可能到第二天）。

### 输入描述:
输入一个整数 \(n(0 \leq n \leq 23)\)，表示区域赛的开始时间。

### 输出描述:
输出一个整数，表示结束时刻的小时数（\(0 \leq\) 答案 \(\leq 23\)）。

### 示例 1
- **输入**
```
9
```
- **输出**
```
14
```
- **说明**
对于这组数据，\(9\) 时开始，\(5\) 小时后为 \(14\) 时。

### 示例 2
- **输入**
```
23
```
- **输出**
```
4
```
- **说明**
对于这组数据，\(23\) 时开始，\(5\) 小时后为 \(4\) 时（第二天）。
