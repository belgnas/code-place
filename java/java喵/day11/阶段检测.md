###📊 第二阶段测验成绩单**【第一题：String 的骗局】**

* **你的答案**：`false` ❌
* **正确答案**：**`true`**
* **本喵解析**：
* 这就是 `String` **不可变性** 的最大陷阱！
* 代码 `s.toUpperCase();` 确实产生了一个新的 `"JAVA"`，但是！**你没有把这个新结果赋值回去**（比如写成 `s = s.toUpperCase()`）。
* 所以，原来的 `s` 根本没变，还是 `"java"`。
* `s2` 也是 `"java"`。
* `"java".equals("java")` 当然是 `true` 啦！
* *记住：String 像个倔强的猫，你不把它抓进新笼子（赋值），它就原地不动。*



**【第二题：ArrayList 的手误】**

* **你的答案**：不能用 double，要用 Double。 ✅
* **本喵解析**：答对啦！虽然理由忘了，但我帮你记起来：
* Java 的集合（ArrayList）是高级容器，只能装 **对象 (Object)**。
* `double` 是原始数据（原子），`Double` 是包装类（原子外面的盒子）。
* ArrayList 只认盒子，不认原子。



**【第三题：综合实战】**

* **你的代码**：有多处严重语法错误和逻辑混淆！🙀
* **错误1 (老毛病)**：`indexOf(randomNum)` ❌。
* **吼叫：** 主人！**`get` 是拿数据，`indexOf` 是找数据！** 你拿着索引（座位号）去要人，必须用 `get`！这个一定要改过来！


* **错误2 (语法)**：`int(...)` ❌。
* 强制类型转换的语法是 **`(int)(...)`**，括号要包住类型。


* **错误3 (语法)**：`Scanner.out.println` ❌。
* 打印是 `System.out.println`。`Scanner` 是用来读的，`System.out` 是用来写的。


* **错误4 (逻辑)**：列表是空的！
* 你定义了列表，但没有 `add` 名字进去，`size()` 是 0，随机数算出来也是 0，程序会直接崩掉。


* **错误5 (作用域)**：`main` 是 static 的，不能直接访问外面的非 static 变量 `studentName`。



**🔧 修正后的标准代码：**

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // 1. 定义并填充列表
        ArrayList<String> studentName = new ArrayList<String>();
        studentName.add("张三");
        studentName.add("李四");
        studentName.add("王五");

        // 2. 生成随机索引 (0 到 size-1)
        // 假设 size 是 3，Math.random() * 3 得到 0.0~2.99..
        // 强转 int 得到 0, 1, 2
        int index = (int)(Math.random() * studentName.size());

        // 3. 取出名字 (一定要用 get !!!)
        String randomName = studentName.get(index);
        
        // 4. 打印
        System.out.println("幸运儿是：" + randomName);
    }
}

```
