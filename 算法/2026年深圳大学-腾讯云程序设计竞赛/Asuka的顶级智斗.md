# **【算法题】Asuka的顶级智斗**

**时间限制**: C/C++/Rust/Pascal 1秒，其他语言2秒  
**空间限制**: C/C++/Rust/Pascal 256 M，其他语言512 M  
**Special Judge, 64bit IO Format**: %lld

## 题目描述

Asuka 最近非常无聊，于是她委托美里小姐帮她买一些漫画。要强而又聪明的她自然不可能去看一些没有营养的无聊漫画，她选择的是一部第二次冲击前连载的智斗漫画，剧情大概是这样的：

夜神月和 L 是一对老对手。有一天，他们突然都获得了 Death note，使用方法是把人的名字写在上面就可以抹除那个人。他们想干掉对面，然而双方都十分谨慎，所以他们都不知道对方的真名。已知两人的真名都是一个整数，夜神月的真名为 $x$，L 的真名为 $y$。

他们想要知道两人数字之和以便于反推出对方的真名，于是经过调查，他们都知道了两个数 $s_1$ 和 $s_2$。但由于两人互相玩操作，导致他们都不知道究竟哪个数才是两数之和。他们只知道至少有一个数是两数之和，且 $s_1$ 和 $s_2$ 比他们任意一个人的数字都要大。

更正式地，满足 $\min(s_1, s_2) > \max(x, y)$，且 $s_1 = x + y \ \lor \ s_2 = x + y$。

两个人轮流对决，且只有一次下笔机会。每个回合该方只能表示不知道或者写下对方真名。若表示不知道则游戏继续，且双方只有在完全确定对方数字的时候才能在 Death note 上写上对面的真名从而获胜。L 的道德水平相对较高，因此夜神月获得先手。

由于第二次冲击带来的动荡与经济衰退，这部漫画已经断更好久了，Asuka 等得十分不耐烦。她很想知道，按照这个规则，且已知夜神月和 L 都会遵循最优策略行动，那么最后谁会赢？

## 输入描述

输入第一行包含一个整数 $T$，代表有 $T$ 组测试样例（$1 \le T \le 10^5$）。

接下来 $T$ 行，每行包含四个整数 $x, y, s_1, s_2$（$1 \le x, y \le 10^9, 2 \le s_1, s_2 \le 2 \times 10^9$），含义如题目所述。

## 输出描述

输出共 $T$ 行，每行包含一个字符串，若夜神月获胜输出 `Yagami`，若 L 获胜输出 `L`。

## 示例 1

**输入**

```
3
3 3 6 8
4 7 11 8
1 2 3 3
```

**输出**

```
Yagami
L
Yagami
```

## 示例 2

**输入**

```
2
7 3 9 10
1 4 5 14
```

**输出**

```
Yagami
Yagami
```

## 备注

故事中，两人的真名一样，都是 $3$。他们经过不懈调查都获得了两个数，$6$ 和 $8$。显然，$6$ 才是真正的两数之和，但他们目前还无从分辨。

夜神月和 L 带着这些信息，即将进入终章。
